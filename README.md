# atividade 

## descrição
este repositório contém a resolução dos exercícios de lógica de programação da aula 05.

## questões resolvidas

a: número positivo ou negativo  
b: dobro do número  
c: valor e tipo da variável  
d: par ou ímpar  
e: triplo do número  
f: maior entre dois números  
g: maior que 10  
h: raiz quadrada  
i: metade do valor  
j: intervalo de 0 a 10  
k: par positivo, par negativo ou ímpar  
l: soma e comparação  
m: maior que 100  
n: múltiplo de 3  
o: entre 10 e 20  
p: tipo e quadrado  
q: classificação por idade  
r: classificação completa do número  
s: números iguais ou diferentes  
t: intervalo de 0 a 100  

## forma de executar o código

baixe o arquivo `exercicio_00.py`  
abra o terminal  
execute o comando:  
`python exercicio_00.py`
